- clonar repositorio: git clone "url"
- iniciar repositorio local: git init
- adicionar arquivos: git add <nome do arquivo>
- adicionar todos arquivos da pasta: git add .
- commitar alterações: git commit -m "mensagem"
- link com repositorio remoto: git remote add origin "url do repo remoto"
- enviar alteracoes para repo remoto: git push -u <remote> <branch>
- recuperar alteracoes de repo remoto: git pull
- criar nova branch: git branch <nome da branch>
- ir para outra branch: git checkout <nome da branch>
- mergear com branch: git merge <nome da branch para mergear>
- mostrar status: git status

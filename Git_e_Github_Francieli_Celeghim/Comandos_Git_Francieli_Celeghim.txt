1. Escrever os comandos mais usados até o momento, com a sintaxe e explicações do
que fazem. Use como referência o seguinte modelo e os exercícios realizados na aula
de Terminal:

git init Crie um novo repositório;
git status (mostra o status dos arquivos em seu repositório);
git add nome_arquivo_diretorio adicione (arquivo específico);
git add . / git add --all  adicione (todos os arquivos);
git commit nome_arquivo -m "enviar mensagem" Confirme um arquivo/diretório;
git push -u origin main (o envio para o repositório deve conter seu nome
e branch);
git pull (atualiza arquivos em relação à branch atual) Atualize o repositório local com base no repositório remoto;
git clone git@github.com:meunome/arquivo-git.git Clone um repositório remoto existente;
